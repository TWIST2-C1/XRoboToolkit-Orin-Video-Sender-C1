## まとめ
このリポジトリは **Jetson Orin** 上で **ZED カメラまたは USB Webcam** の映像を **ハードウェア H.264/HEVC エンコード** し、**TCP / UDP / ASIO / ZeroMQ** のいずれかで外部デバイスへリアルタイム配信するツールキットです。各ファイルは **「何を目的としているか」** と **「どのように実現しているか」** が明確に分離されており、通信方式やカメラ種別を選択的に組み合わせて拡張しやすい構造になっています。
